# 🌐 Responsive Tailwind Website Template

A simple and responsive website template built using **HTML** and **Tailwind CSS**.
This project demonstrates how to design a clean modern webpage with a navigation bar, hero section, image gallery, contact form, and footer.

---

## ✨ Features

* 📱 **Responsive Layout**
* 🧭 **Navigation Bar**
* 🏞️ **Hero Section with Background Image**
* 🖼️ **Image Gallery**
* 📩 **Contact Form**
* 🔗 **Social Media Links**
* 🎨 **Styled with Tailwind CSS**

---

## 🛠 Technologies Used

* **HTML5**
* **Tailwind CSS (CDN)**

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder in **VS Code** or any code editor.
3. Open `index.html` in your browser.

That’s it! The website will run locally.

---

## 📸 Sections in the Website

* **Navigation Bar** – Provides links to important sections.
* **Hero Section** – Welcome message with a background image.
* **Image Gallery** – Displays multiple images in a grid layout.
* **Contact Form** – Allows users to send a message.
* **Footer** – Contains copyright and social media links.
